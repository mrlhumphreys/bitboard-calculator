# Bitboard Calculator

A bitboard is a data structure used for the efficient storage of board game state. This applies to board games that represent their board as a 2-d grid.

This tool for converting a visual representation of a bitboard into binary.

## Quickstart Instructions

 1. Visit [Bitboard Calculator](https://mrlhumphreys.github.io/bitboard-calculator)
 2. Specify the size of the board and click submit. (Default 8x8)
 3. Click on the grid to switch on/off the cell state.
 4. Binary and Decimal outputs automatically reflect board state.

## Dev Instructions

 * Start the server by running `./scripts/start.sh`. (Requires ruby installed)
 * Static assets (index.html) is in the `./public` directory.
